# Contributing

Thanks for your interest in contributing to `@aluvia/sdk`.

## Requirements

- Node.js 18+

## Local setup

Install dependencies:

```bash
npm ci
```

## Build

```bash
npm run build
```

This produces:

- `dist/esm` (ESM)
- `dist/cjs` (CJS)
- `dist/types` (TypeScript declarations)

## Test

```bash
npm test
```

## Pull requests

- Keep changes small and focused.
- Add or update tests for behavior changes.
- Avoid committing secrets. Use environment variables for API keys.
