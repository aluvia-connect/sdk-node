# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability in `@aluvia/sdk`, please report it privately.

- Email: `security@aluvia.io`
- Include: a description of the issue, impact, reproduction steps, and any suggested fix or mitigation.

Please do not open a public GitHub issue for security reports.

## Response Timeline

We aim to acknowledge reports within 2 business days and provide a remediation plan or workaround as quickly as possible.
